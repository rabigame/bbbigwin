# 🎰 BIGWIN Images

BBDiscord システム用の画像ストレージリポジトリ

## 概要

このリポジトリは、BBDiscordシステムの通知で使用される画像ファイルを保存するために使用されます。
画像はGitHub Pagesを通じて配信されます。

## 画像URL形式

```
https://rabigame.github.io/bbbigwin/upload/images/[画像ファイル名]
```

## ディレクトリ構造

```
bbbigwin/
└── upload/
    └── images/         # 画像保存先
        ├── bigwin_*.png
        ├── bigwin_*.jpg
        └── bigwin_*.webp
```
